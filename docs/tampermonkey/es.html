YouTube CleanFeed

Userscript de Tampermonkey para limpiar YouTube.

Descripción

YouTube CleanFeed oculta automáticamente contenido no deseado en YouTube.

  • Shorts
  • Noticias en la página principal
  • Vídeos con pocas visualizaciones
  • Opción para bloquear Shorts

Instalación

  1. Instalar Tampermonkey
  2. Instalar el script desde GreasyFork
