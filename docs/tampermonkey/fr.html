YouTube CleanFeed

Userscript Tampermonkey pour nettoyer le flux YouTube.

Description

YouTube CleanFeed est un userscript Tampermonkey qui masque automatiquement les contenus indésirables sur YouTube.

Il masque :

  • Shorts (Accueil, Recherche, barre latérale)
  • Actualités / News sur l’accueil
  • Vidéos peu populaires
  • Option : bloquer la page Shorts (redirection vers l’accueil)

Fonctionne sur :

  • Page d’accueil
  • Recherche
  • Recommandations des vidéos
  • Navigation SPA de YouTube (sans rechargement)

Fonctionnalités

  • Seuil de vues personnalisable
  • Dock de contrôle flottant (replier / déplier)
  • Modes de performance (Performance / Éco / Ultra Éco)
  • Masquage automatique du header sur les pages vidéo
  • Notifications toast
  • Optimisations anti-flicker et performances

Installation

  1. Installer Tampermonkey
  2. Installer le script depuis GreasyFork
  3. Ouvrir YouTube et ajuster les options si nécessaire

Compatibilité

  • Tampermonkey
  • YouTube Desktop
  • Chromium & Firefox
