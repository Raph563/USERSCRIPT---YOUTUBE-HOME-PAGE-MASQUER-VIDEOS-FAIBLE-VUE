YouTube CleanFeed

Tampermonkey userscript to clean the YouTube feed.

Description

YouTube CleanFeed is a Tampermonkey userscript that automatically hides unwanted content on YouTube.

It hides:

  • Shorts (Home, Search, sidebar)
  • News / Top stories on Home
  • Low-view / low-popularity videos
  • Optional: block the Shorts page (redirect to Home)

Works on:

  • Home page
  • Search
  • Watch recommendations
  • YouTube SPA navigation (no reload)

Features

  • Custom view threshold
  • Floating control dock (collapse / expand)
  • Performance modes (Performance / Saver / Ultra Saver)
  • Auto-hide header on watch pages
  • Toast notifications
  • Performance and anti-flicker optimizations

Installation

  1. Install Tampermonkey
  2. Install the script from GreasyFork
  3. Open YouTube and adjust settings if needed

Compatibility

  • Tampermonkey
  • YouTube Desktop
  • Chromium & Firefox
