YouTube CleanFeed

Userscript Tampermonkey para limpar o feed do YouTube.

Descrição

YouTube CleanFeed é um userscript Tampermonkey que oculta automaticamente conteúdos indesejados no YouTube.

Oculta:

  • Shorts (Página inicial, Pesquisa, barra lateral)
  • Notícias / Top stories na página inicial
  • Vídeos com poucas visualizações
  • Opcional: bloquear a página Shorts (redirecionar para a Home)

Funciona em:

  • Página inicial
  • Pesquisa
  • Recomendações de vídeos
  • Navegação SPA do YouTube

Funcionalidades

  • Limite de visualizações personalizável
  • Dock de controle flutuante
  • Modos de desempenho
  • Ocultar cabeçalho automaticamente na página de vídeo
  • Notificações toast
  • Otimizações de desempenho e anti-flicker

Instalação

  1. Instale Tampermonkey
  2. Instale o script via GreasyFork
  3. Abra o YouTube e ajuste as opções

Compatibilidade

  • Tampermonkey
  • YouTube Desktop
  • Chromium & Firefox
